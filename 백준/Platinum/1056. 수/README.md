### 문제 설명

<ol>
	<li>현재 수를 1 증가시킴 (현재 수+1)</li>
	<li>현재 수가 1보다 크면 1 감소 시킴 (현재 수-1)</li>
	<li>현재 수를 자연수 제곱함. (현재 수^x, x는 자연수)</li>
</ol>
